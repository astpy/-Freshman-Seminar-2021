Q. Pythonって何ができるんですか？
A. AIの開発やWeb上で動くアプリケーションの作成が比較的簡単にできます。
Pythonは数値計算(四則演算や微積分など)に長けているのでAIの開発に向いていたり、ライブラリやAPIが非常に多く用意されているのでさまざまなアプリケーションが開発できたりします。
ライブラリ -> 先人たちが複雑な操作を簡単に書けるように用意してくれた、スターターキットのようなもの。Pythonで言うと import xxx でライブラリを使えます。
API -> Application Programming Interfaceの略。外部のアプリケーションの機能を簡単に利用できるように用意されているもの。Pythonでは、YouTubeやGoogle Spreadsheets、Twitter、ぐるなびなど、様々なサービスがPython用のAPIを用意しています。

Q. Pythonって2と3がありますが、どちらのほうがいいんですか？
A. Python3を使うようにしましょう。
Python2は公式サポートが終了していますし、新しい機能はPython3に追加されていきますので、Python3のほうが圧倒的におすすめです。
